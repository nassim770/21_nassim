    Intro

    HTML es el lenguaje de marcado fundamental de la web: estructura y significado del contenido (títulos, párrafos, imágenes, enlaces, listas, etc.). No es un lenguaje de programación; no genera lógica por sí mismo. Con HTML5 se favorece la semántica y la accesibilidad usando etiquetas que describen el contenido.

    Desarrollo

    Estructura básica: El DOCTYPE para HTML5 se escribe al inicio para indicar que se está usando HTML5. El documento se envuelve en una etiqueta contenedora que se llama html. Etiquetas y elementos comunes: El título de la página, que aparece en la pestaña, se define con la etiqueta title dentro del head. Encabezados de diferentes tamaños se crean con h1 a h6, y el texto se agrupa con p para párrafos.

    Fin

    HTML es la base de cualquier página: define estructura y significado del contenido. HTML5 mejora semántica, accesibilidad y capacidades multimedia sin depender solo de otras tecnologías.
